# Zubeda Hassan Sakuru — Personal Portfolio

This repository contains the **source code for personal portfolio website**, built using **HTML, CSS, and JavaScript**.
The portfolio highlights  professional profile, projects, testimonials, blog posts, and contact information in a clean and responsive design.

 **Live Website:** [https://zubedasakuru.vercel.app](https://zubedasakuru.vercel.app)

---

##  About the Website

This portfolio website is designed to:

* Showcase her **professional experience** and leadership role as **CEO of Simba Sports Club, Tanzania**
* Present a **visual portfolio of projects and achievements**
* Display **testimonials** from collaborators and clients
* Provide **blog insights** on industry trends and personal experiences
* Offer an easy way for visitors to **contact her**

The website is fully responsive, works on all devices, and is optimized for performance and usability.

---

##  Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (vanilla)
* **Styling:** Custom CSS with responsive layouts
* **Deployment:** Vercel
* **Version Control:** Git & GitHub

---

##  Features

* Home section with professional introduction and image
* About section highlighting biography and leadership experience
* Portfolio section showcasing projects and achievements
* Testimonials section featuring client/partner feedback
* Blog section with articles and updates
* Contact section with a functional contact form
* Fully responsive for mobile and desktop devices


##  Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. **Navigate to the project folder:**

```bash
cd your-repo-name
```

3. **Open `index.html` in your browser** to view the website locally.

The website is **static**, so no server setup or dependencies are required.

---

##  Customization

To update the portfolio:

* Edit HTML files (`index.html`, `about.html`, `portfolio.html`, etc.) for content
* Replace images in the `images/` folder
* Update styles in `css/style.css`
* Modify JavaScript behaviors in `js/script.js`
